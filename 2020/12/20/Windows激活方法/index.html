Windows激活方法
2020-12-20 14:45:22 # Windows

作为一个安全专业的，在使用破解版的Windows系统或者使用网络上现有的一些破解工具时总是不太放心。但是苦于没钱买正版，就只能使用一种相对的比较靠谱一点的激活方式。

Windows 7

Win7的话，我曾经找到过一个可以用的激活码，不知道现在还能不能用。

RHTBY-VWY6D-QJRJ9-JGQ3X-Q2289

Windows 10

Win10与之前的与之前的windows版本一样，有家庭版、专业版、企业版等各个版本。Win10的激活方式也有Retail零售版激活、OEM版激活、VOL版激活（包括KMS激活等方式）等方式。其中OEM是根据主板信息自动联网激活，属于永久激活的，而KMS激活是180天自动循环，属于未购买win10的用户用的比较多的一种激活方式，许多激活软件就是依据KMS激活的原理，如KMSpico,小马激活等。

我这里使用的也是KMS激活方式，只不过用的密钥管理服务器是搭建在自己VPS上的。

在Win10桌面状态下，右键Windows图标或按快捷键win+x，点击命令提示符（管理员）。

第一条命令，slmgr.vbs /upk ，是用于卸载当前电脑的密钥

第二条命令，slmgr /ipk xxxx，其中xxxx是密钥，一般用上面的密钥即可

第三条命令，slmgr /skms zh.us.to ，是设置kms服务器地址，一般很多kms地址失效，可以尝试其他地址，如kms.xspace.in，kms.shuax.com、skms.ddns.net 等等

第四条命令，slmgr /ato ，是激活命令，连接到kms服务器进行windows激活。

P.S.

slmgr.vbs -dlv:这条命令可以查看详细的激活信息。

slmgr.vbs -xpr:这条命令可以查看过期时间。

再说下关于数字权利激活（OEM激活）。前面提到，这种激活是永久的，而且是绑定主板信息的。也就是说，如果更换了电脑硬盘，内存条，网卡等硬件，只要主板在，全新安装Win10后（版本要对应上），联网即可激活。
